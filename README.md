# SignalsHQ Product-Led Redesign Concept

Product-led homepage redesign built as a fast-track application for the Frontend Product Engineer Intern role.

## Idea

SignalsHQ helps CPA firms use AI across tax workflows. This redesign concept makes that product promise visible immediately by showing a realistic command-center experience: client intake, document extraction, tax research, review queues, and audit-ready outputs.

## What This Shows

- A CPA-focused product narrative, not just a visual refresh.
- Responsive React UI with interactive workflow tabs, product modules, and a simulated AI tax assistant.
- Clear attention to trust, review workflows, audit trails, and financial-data sensitivity.
- Product judgment around an AI-native B2B workflow.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
```
